List users with or without filtering

List users with or without filtering

path Parameters
uid
required
string

Tenant's unique identifier

orgUid
required
string

Organization's unique identifier

query Parameters
filter
string
Example: filter=firstName eq 'Charles' and lastName eq 'BABBAGE'

Search filter expression: property $operator '$value' where operator is one of:

  • eq: equals
  • ne: not equals
  • gt: greater than
  • ge: greater or equal
  • lt: lower than
  • le: lower or equals
  • lk: like

Allowed properties:

  • firstName
  • lastName
  • email
  • login
  • uid or uuid

Value must be enclosed with single quotes

For more complex queries enclose with parenthesis and use standard operands (and, or, not)

Example: (firstName eq 'Charles') and ( (lastName eq 'BABBAGE') or (lastName eq 'DARWIN'))

Note: URL length must not exceed 2083 otherwise a 414-URI Too Long is returned

index
integer

Page index

size
string
Example: size=+lastName

Page size

orderBy
any

Order by expression.

Use + sign for ascending sorting and - sign for descending sorting

User comma as multiple values separator

example: +firstName,-lastName sorts by first name ascending and the by last name descending
